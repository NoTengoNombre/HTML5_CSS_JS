La pseudo-clase de idioma :lang()

La pseudo-clase de idioma :lang() permite especificar
el aspecto de los elementos de un idioma determinado.

En el ejemplo siguiente se cambia las comillas utilizadas
en una cita < q> según el idioma indicado en el atributo lang.
