Propiedad opacity

Permite hacer que un elemento sea parcialmente transparente.

El valor de esta propiedad debe ser un valor decimal entre 0 y 1

(El valor 0 hace que el elemento
sea completamente transparente
y el valor 1 hace que el elemento
sea completamente opaco)
