Interlineado : line-height

Line-height

Permite establecer el interlineado de un bloque de texto.

El valor se puede expresar en cualquier unidad de longitud, aunque lo más lógico es utilizar porcentajes o em.

Si se expresa en porcentajes, el valor 100% corresponde al interlineado normal.
