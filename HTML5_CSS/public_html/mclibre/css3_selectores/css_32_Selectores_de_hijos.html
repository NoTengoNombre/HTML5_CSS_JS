Se escriben dos etiquetas seguidas y separadas por un signo
"mayor que" (E > F), las propiedades afectan
a los elementos con la segunda etiqueta (F)
contenidos dentro de la primera etiqueta (E),
pero no afecta si hay etiquetas intermedias entre ellos.

Unicamente los párrafos
< p> dentro de una división < div>
sin elementos intermedios se ven de color rojo.
