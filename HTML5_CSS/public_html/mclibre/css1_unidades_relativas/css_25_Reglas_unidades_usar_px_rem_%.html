Unidades recomendadas

Valores a usar

px - para distancias pequeñas (anchos de bordes, márgenes, etc.).

rem - para tamaños de fuente, salvo que se quiera aprovechar
en el diseño la composición de tamaños, en cuyo caso se usarán porcentajes %.

porcentajes - % para tamaños de elementos grandes y
distribución de elementos por la pantalla.
