La siguiente división, cuyo tamaño está establecido por la hoja de estilo, tiene una imagen de fondo situada en el centro a la derecha

Solo 1 parametro : background-position: 100%;

La recomendación CSS 2 del W3C no permitía mezclar números y palabras, pero la recomendación CSS 2.1 sí que permite mezclar números y palabras, aunque obligan a que el primer término se refiera a la posición horizontal y el segundo a la vertical. Los navegadores muestran la imagen correctamente, abajo a la derecha.

esquina_derecha : background-position: 100% bottom
centro_superior : background-position: 50% top;
esquina_superior derecha : background-position: 100% top
izquierda_superior : background-position: 0% top;
centro : background-position: 50% center;
