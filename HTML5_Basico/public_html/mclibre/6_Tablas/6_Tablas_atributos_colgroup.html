Para indicar el número de columnas que
forman parte de un grupo de columnas,
se pueden utilizar etiquetas < col />
(con o sin atributo span) o el atributo
span de < colgroup >.

Si se utiliza el atributo span,
su valor indica el
número de columnas que forman
parte del grupo de columnas.
Pero si se utiliza el atributo span
en la etiqueta < colgroup />,
no se pueden incluir etiquetas
< col /> en el < colgroup />.
