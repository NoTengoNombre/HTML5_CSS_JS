Imágenes de fondo múltiples varios valores

Tiene dos imágenes de fondo.
El orden de las imágenes es de delante hacia atrás, es decir, la primera imagen de la lista está por encima de la segunda y así sucesivamente.
El orden es importante pues si una imagen no tiene transparencia tapará las siguientes.
