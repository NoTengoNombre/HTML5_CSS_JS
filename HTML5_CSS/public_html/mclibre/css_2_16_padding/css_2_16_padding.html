El margen interior

Un elemento es el espacio transparente
situado entre el elemento y el borde.
Este espacio se puede de definir
aunque el elemento no tenga borde.
La hoja de estilo por defecto del navegador
aplica márgenes interiores a muy pocos elementos.

  • La propiedad padding permite establecer el tamaño del margen interior.
    Como en el caso de las propiedades
    border-color, border-width y border-style
    se pueden escribir de uno a cuatro valores, que se interpretan de la siguiente manera:
  • 1 valor: este valor se aplica a los cuatro lados
  • 2 valores: el primer valor se aplica a los lados inferior y superior y el segundo a los lados derecho e izquierdo
  • 3 valores: el primer valor se aplica al lado superior, el segundo a los lados derecho e izquierdo y el tercer valor al lado inferior
  • 4 valores: el primer valor se aplica al lado superior, el segundo al lado derecho, el tercer valor al lado inferior y el cuarto al lado izquierdo
