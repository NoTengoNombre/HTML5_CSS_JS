Posición del marcador: list-style-position

La propiedad list-style-position establece la posición del marcador con respecto al resto del elemento de la lista:
  • dentro (inside)
  • fuera (outside)
Los navegadores aplican por omisión el valor outside.
  • Posición del marcador: list-style-position - outside
  • Posición del marcador: list-style-position - inside
