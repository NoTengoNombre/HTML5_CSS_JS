Position : relative

Si se utiliza la propiedad z-index y posicionamiento,
los elementos con mayor z-index se colocan
sobre los elementos con menor z-index.

Los elementos sin z-index se tratan
como si tuvieran un z-index igual a 0.

Primer parrafo
position: relative;
top: 150px;
left: 50px;
z-index: 3

Segundo parrafo
position: relative;
top: -150px;
left: 0px;
z-index: 1

Tercer parrafo
position: relative;
top: -150px;
left: 0px;
z-index: 4

Cuarto parrafo
position: relative;
top: -150px;
left: 0px;
z-index: 2
