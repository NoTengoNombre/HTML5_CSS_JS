Para indicar el número de columnas que forman parte de un grupo de columnas,
se pueden utilizar etiquetas < col /> (con o sin atributo span)
o el atributo span de < colgroup >.

Si se utiliza el atributo span,
su valor indica el número de columnas que
forman parte del grupo de columnas.

Pero si se utiliza el atributo span
en la etiqueta < colgroup />,
no se pueden incluir etiquetas < col /> en el < colgroup />.

Uso 3 espacios para el margen superior

colgroup span 2 - 1 col

Leyenda
1 2 3
4 5 6

Sin el atributo span - 2 colgroup 2 col

Leyenda
1 2 3
1 2 3

Con el atributo colgroup 2 - col span 2

Leyenda
1 2 3
1 2 3
1 2 3
