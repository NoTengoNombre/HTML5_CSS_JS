La propiedad background-origin

La propiedad background-origin permite definir la zona en la que se sitúa el origen de la posición del fondo.

Hay que tener en cuenta que una vez definido el origen de la imagen, esta puede repetirse en todo el esapcio disponible
(al que afecta la propiedad background-clip).

  • border-box: el fondo se extiende hasta el borde (borde incluido)

  • padding-box: el fondo se extiende hasta el margen interior (el borde no se incluye)

  • content-box: el fondo se limita al contenido (no se incluye ni el margen interior ni el borde)
