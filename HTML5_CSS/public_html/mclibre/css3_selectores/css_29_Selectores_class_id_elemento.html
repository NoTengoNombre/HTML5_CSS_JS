E : elemento


E.e : elemento.class


E#e : elemento.id

Selectores de tipo: E, E.e y E#e

Etiqueta (E),
Las propiedades afectan a todos los elementos con etiqueta E.

Escribe una etiqueta seguida de un punto y un nombre de clase (E.e),
las propiedades afectan a todos los elementos con etiqueta E
cuyo atributo class tenga el valor e.

Escribe una etiqueta seguida de una almohadilla
y un nombre de id (E#e), las propiedades afectan
al elemento con etiqueta E cuyo atributo id tenga
el valor e.

Atributo id no se puede repetir, no puede haber dos elementos con el mismo valor
del atributo id (independientemente de que los
elementos tengan etiquetas iguales o distintas).
