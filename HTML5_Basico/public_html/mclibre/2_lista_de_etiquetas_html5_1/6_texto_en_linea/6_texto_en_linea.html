Abreviatura
La etiqueta < abbr > está pensada para
etiquetar abreviaturas o acrónimos.

Si la etiqueta incluye el atributo title,
este debe contener el significado de la abreviatura.

Firefox resalta los elementos < dfn > con atributo title
con un subrayado punteado (propiedad CSS 3 text-decoration:
underline dotted black), pero Internet Explorer y Google Chrome no lo hacen.

n + yn = zn

Tecnicismo:

La etiqueta < i > está pensada para identificar texto con un significado especial , como nombres de especies, términos técnicos, expresiones extranjeras, etc.

Atención:
La etiqueta < b > está pensada para identificar texto sobre el que se quiere
llamar la atención, pero sin que eso signifique que es importante,
como los términos clave en un resumen, nombre de productos comerciales, etc
