Los atributos name no se deben repetir,
porque no se recibiría toda la información al enviarla.
La única excepción es el botón radio,

El atributo value permite establecer el valor inicial de un control
aunque cada control lo utiliza de una forma ligeramente distinta
y algunos controles no lo admiten (área de texto, archivo o imagen).

El atributo required permite indicar qué controles es obligatorio rellenar para enviar el formulario.
