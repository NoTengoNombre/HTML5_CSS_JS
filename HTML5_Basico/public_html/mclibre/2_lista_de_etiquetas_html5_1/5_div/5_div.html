Define una división.
  Esta etiqueta permite agrupar varios elementos de bloque 
  (párrafos, encabezados, listas, tablas, divisiones, etc). 
  En principio, los navegadores no muestran nada especial 
  cuando se crea una división, 
  salvo que se dé formato a la división con la hoja de estilo.
