La propiedad clear hace que un elemento no tenga elementos flotantes a su lado.
Los posibles valores de clear son:

  • left, hace que no hayan elementos flotantes a la izquierda,
  • right, hace que no hayan elementos flotantes a la derecha,
  • both, hace que no hayan elementos flotantes ni a derecha ni a izquierda,
  • none, permite que hayan elementos flotantes a derecha y a izquierda (valor por omisión).

El logotipo de HTML5 debe "flotar" a la izquierda y el párrafo debe fluir a su derecha.

El logotipo de CSS3 a la izquierda y el párrafo debe fluir a su derecha.
Seguramente tanto la imagen como el párrafo están a la derecha del logotipo de HTML5.
