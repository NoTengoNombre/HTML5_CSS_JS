Los navegadores repiten La imagen de fondo tanto en vertical como en horizontal.

capa1 - no-repeat -
La imagen no se repite
capa2 - repeat-x -
La imagen se repite únicamente en horizontal
capa3 - repeat-y -
La imagen se repite únicamente en vertical

CSS 3: Fondos y bordes

capa 4 - space
La imagen se repite en horizontal y vertical, pero se añaden espacios entre las imágenes para mostrar Las imágenes completas y que no se corten en los bordes derecho o inferior.
capa 5 - round
La imagen se repite en horizontal y vertical, pero las imágenes se deforman para mostrarlas completas y que no se corten en los bordes derecho e inferior.
