La propiedad border-collapse permite elegir el modo de presentación de los bordes de las celdas y de la tabla.

Los dos valores permitidos son separate (separado) y collapse (colapsado).

En el modo separado existe un hueco entre los bordes de cada celda, mientras que en el modo colapsado los bordes se superponen.

Si no se establece el modo de presentación,
los navegadores aplican el valor separate.

La propiedad border-spacing
Permite establecer una separación
entre las celdas cuando
se utiliza el modo de bordes separado.

Se puede escribir un único valor,
que se aplicaría a los cuatro lados,
o dos valores, que se aplicarían en
horizontal (a derecha e izquierda)
en vertical (arriba y abajo) respectivamente.

Los valores no se pueden expresar en porcentajes,
pero sí en cualquier otra unidad (px, em, etc).
