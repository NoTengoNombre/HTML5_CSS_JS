Selectores universales: *, . y #

Si se escribe un asterisco (*), las propiedades afectan a todos los elementos de la página.

Selectores de tipo: E, E.e y E#e

Para definir una clase que afecte a cualquier etiqueta,
se debe escribir un punto y el nombre de la clase.

Para definir un id que afecte a cualquier etiqueta,
se debe escribir un punto y el nombre de la clase

El atributo id no se puede repetir
, es decir, que no puede haber en una página web
dos elementos con el mismo valor del atributo id
(independientemente de que los elementos tengan etiquetas iguales o distintas).

En el caso de que la misma hoja de estilo se enlace desde varias páginas web,
en cada una de esas páginas sí que puede haber un elemento
(del mismo tipo o distinto en cada página) con el mismo valor del atributo id.
