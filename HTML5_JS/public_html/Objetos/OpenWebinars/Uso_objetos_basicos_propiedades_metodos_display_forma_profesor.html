Texto resumen de este articulo

La propiedad 'style' desde JS lee las propiedades que se han definido en el 'CSS'

Este esto se ampliará cuando se pulse porque cambiará el estado de la propiedad 'style.display' del CSS mediante JS , de un principio esta propiedad no esta definida. Lo cual dará error ya que se esta invocando a un valor que no existe.
