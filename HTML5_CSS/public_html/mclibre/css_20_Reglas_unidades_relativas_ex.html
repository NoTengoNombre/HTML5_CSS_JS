La unidad relativa em se puede relacionar directamente con los porcentajes, concretamente 1em = 100%.

Unidades relativas em, ex y ch
se produce también el fenómeno de composición

Si se cambia el tamaño en dos elementos anidados,
los porcentajes se multiplican entre sí

conveniente y puede provocar cambios de

no previstos.
