Otra forma de conseguir un menú horizontal es utilizar la
propiedad display: inline
en los elementos de la lista < li >
en vez de la propiedad float.
De esta manera se puede incluso centrar la lista, mediante la propiedad text-align: center
en la lista (< ul > o < ol >).
Otra ventaja es que no hace falta utilizar la propiedad clear.
