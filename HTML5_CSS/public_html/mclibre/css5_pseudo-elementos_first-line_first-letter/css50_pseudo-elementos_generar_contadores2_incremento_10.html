Se pueden generar contadores en los pseudo-elementos
:after y :before mediante la propiedad content
y el valor counter(nombre_de_contador).


El contador debe ponerse a cero con la
propiedad counter-reset y aumentarse con la propiedad counter-increment.

Este contador se genera al principio de cada párrafo,
se pone a cero con el elemento < pre> y se incrementa en cada párrafo.
