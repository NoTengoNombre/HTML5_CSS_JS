Varias etiquetas seguidas (E F G ... )

Si se escriben varias etiquetas seguidas (E F G ... ),
las propiedades afectan a los elementos con la última
etiqueta contenidos dentro de la etiqueta anterior,
contenidos a su vez dentro de etiqueta anterior y
así sucesivamente, aunque haya etiquetas intermedias.

Los párrafos dentro de un elemento de lista < li>
dentro de una división < div> se ven de color rojo.
