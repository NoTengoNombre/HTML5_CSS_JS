Testing vmin

vmin: viewport min

Porcentaje de la medida actual del viewport,
pero se calcula con respecto al lado más corto.

Es decir que si el ancho del viewport es más
corto que el alto, entonces vmin será lo mismo
que utilizar vw.
Por el contrario, si el alto
del viewport es más corto que el ancho, entonces vmin será lo mismo que vh. = viewheight
