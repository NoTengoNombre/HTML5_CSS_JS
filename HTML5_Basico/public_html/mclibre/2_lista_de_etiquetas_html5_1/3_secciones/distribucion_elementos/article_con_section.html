Las etiquetas <article> y <section> son similares.
La diferencia entre ellas es que <article> es para partes
que forman una unidad en sí mismas y <section> es para partes
de una unidad mayor.
Si la página trata de un único tema y
se va a utilizar un único elemento <article>,
es preferible utilizar la etiqueta <main>.
